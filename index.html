Sambung ke Google Sheets
Masukkan URL Web App Apps Script anda untuk menyambungkan sistem ini.
Cara dapatkan URL Web App:
  1. Buka Google Sheet anda → Extensions → Apps Script
  2. Paste kod dari fail google_apps_script.js yang disertakan
  3. Klik Deploy → New deployment → Web App
  4. Execute as: Me · Who has access: Anyone
  5. Klik Deploy → salin URL yang terhasil ke sini
URL bermula dengan https://script.google.com/macros/s/ dan berakhir dengan /exec
